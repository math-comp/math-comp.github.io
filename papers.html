This is a list of research results using the Mathematical Components published among other in the following venues:

  • ITP = Interactive Theorem Proving
  • CPP = Certified Programs and Proofs
  • CICM = Conference on Intelligent Computer Mathematics
  • LMCS = Logical Methods in Computer Science
  • ICFP = International Conference on Functional Programming
  • POPL = Principles of Programming Languages
  • PPDP = Principles and Practice of Declarative Programming

This list is non-exhaustive and not up-to-date (it stop at 2018, should be updated soon). Initial contents come from the official wiki.
